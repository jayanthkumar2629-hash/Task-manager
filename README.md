# Task Manager Application

## Overview

A full-stack Task Manager application built using React.js, Node.js, Express.js, MongoDB Atlas, and JWT Authentication.

## Features

* User Registration
* User Login
* JWT Authentication
* Protected Routes
* Add Tasks
* Complete Tasks
* Delete Tasks
* Logout
* Dark Mode UI

## Tech Stack

### Frontend

* React.js
* React Router
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### Authentication

* JSON Web Token (JWT)

## Installation

### Backend

cd server

npm install

npm run dev

### Frontend

cd client

npm install

npm run dev

## Usage

1. Register a new account
2. Login
3. Add tasks
4. Mark tasks as completed
5. Delete tasks
6. Logout
